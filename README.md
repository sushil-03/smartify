# Smartify

## Technologies Used

- React
- TypeScript
- Material-UI
  - Slider
  - Switch
  - Typography
  - Theme (for dark-light mode)
- Responsive Design

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/sushil-03/smartify

   ```

2. Change directory to smartify

```sh
  cd smartify
```

3. Install Dependency

```sh
  npm install
```

4. Start Local server

```
  npm start
```
