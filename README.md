# Smartify

## Technologies Used

- React
- TypeScript
- Material-UI
  - Slider
  - Switch
  - Typography
  - Theme (for dark-light mode)
- Responsive Design

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/yourusername/yourproject.git

   ```

2. cd smartify

3. npm install

4. npm start
